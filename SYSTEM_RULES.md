1️⃣ Session Rules

1- No overlapping sessions allowed

2- Duration auto-calculated from start and end time

3- Unlimited sessions per day

4- Once confirmed, session cannot be edited

5- Deep Work = session ≥ 45 minutes

2️⃣ Mood & Confidence

1- Mood = Interest level before session (1–5 scale)

2- Confidence = Understanding level after session (1–5 scale)

3- Reflection note stored but not used in prediction (V1)

3️⃣ Productive Day Rule

1- A day is considered productive if:

Total study time ≥ 60 minutes
OR

At least one deep work session

4️⃣ Consistency Score

Weekly Consistency =
(Productive Days / 7) × 100

Classification:

80–100% → Strong

60–79% → Moderate

40–59% → Unstable

<40% → High Risk

5️⃣ Prediction Activation

Prediction becomes active only if:

Minimum 14 days of data

Minimum 2 assessments recorded for subject

6️⃣ Prediction Output

Uses entire semester history (V1)

Recalculated weekly

Rounded predicted score (e.g., 67%)

Provides fixed-number recommendations (e.g., 4 sessions/week)

7️⃣ Version 1 Scope Limitations

1 student

1 semester

Maximum 5 subjects

College semester structure only
