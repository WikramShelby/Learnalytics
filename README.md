# Learnalytics
Behavioral Learning Intelligence &amp; Academic Performance Forecasting System

Learnlytics is a behavioral learning intelligence system designed to analyze study patterns, detect subject imbalance, and forecast semester performance for college students.
Unlike traditional study trackers, Learnlytics focuses on:
-Behavioral pattern detection
-Deep work analysis
-Consistency scoring
-Subject-level imbalance detection
-Rounded performance forecasting
-Adaptive, supportive recommendations
